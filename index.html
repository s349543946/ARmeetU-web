产品展示

「AR浅社交平台」

遇·见是一款利用AR技术进行社交互动的手机APP，旨在建立一个以地点为导向的全新浅社交平台。用户可以对某景点发表一些看法，或是几句感慨，此刻的“心情”便会停留在当时那个地点，并在地图上显示一个标记，称为心情标记。本平台着重实现用户“藏心情”与“找心情”的过程，增强用户探索的好奇心，为用户交流创造更多的可能性，同时可以将用户即兴的想法与周边环境很好的结合在一起，打造全新的浅社交模式。

Learn More

「找心情」

用户可以通过地图看到在他周边的心情标记。当他试着点击某个心情标记时，便能显示出所有停留在那个地点的心情概况。心情以被点赞数进行排列。

Learn More

「浏览心情」

当用户满足同一个地方扫描同一个实景时，便能看到心情动态的具体信息。同时，用户之间可以进行点赞、评论、关注等。

Learn More

用户定位

校园推广阶段
面向在校大学生
通过趣味性的“藏心情”与“找心情”过程扩大交友圈
对学校发生的各种新鲜事、趣事进行分享
对课堂教学、美食小吃等表达即兴的所思所感

社会推广阶段
面向旅行爱好者、所有社交爱好者
在旅游时对所见之景、所遇之事进行评论分享
记录旅行足迹、在时空上与其他用户相遇
对商家、美食、住宿进行评论，达到经验共享

周边服务

个性化的周边服务，为您制作足迹时光轴、量身定制足迹相册、推荐旅行计划——“遇见”值得记录的生活

足迹时光轴

本产品提供用户足迹时光轴的服务。时光轴是指将用户发布过的动态以时间串联，并标注地点，显示具体动态内容。记录年华岁月中的故事，时空不再是障碍，只需一条线，就能够回到从前。

足迹相册

本产品为用户提供足迹相册服务，即根据用户的足迹以及发布的动态为用户量身打造一本实体相册，记录用户曾经去过的地方、那时的心情或是想说的话，作为用户生活点滴的记录与反馈。

旅行计划推荐

本产品为用户提供旅行计划推荐的服务，即根据用户的足迹数据预测用户的出行习惯，与旅行社合作，旨在为用户量身打造适合自己的旅行计划。生活不止忙碌的安排，还有静心可待的惊喜。
